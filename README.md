# Midnight 9XM 🎧

A stylish, modern, and responsive web-based music player inspired by Spotify. Midnight 9XM allows users to stream albums, artists, and playlists with a smooth UI and dark theme, built using Flask, Bootstrap 5, and Jinja2.

## 🌟 Features

- 🎵 Stream songs from albums, playlists, or artists
- 🧭 Sidebar navigation (Albums, Genres, Top Charts, etc.)
- 🎧 Dark mode with green accent (Spotify-style)
- 🔁 Horizontal carousels for each category
- 🚀 Play button overlay on hover
- 📱 Mobile responsive UI

## 📦 Setup Instructions

### 1. Clone the repository

```bash

cd midnight-9xm
```
## Create a virtual environment and activate it
